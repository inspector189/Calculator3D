3D Math Calculator

Instruction:

Create vectors:
a = vec3(-100, 10, 100)
b = vec3(200, 200, 12)

Operations:

add     : addVectors     c = a add b

sub     : subVectors     c = a sub b

mul     : mulVector     c = a mul 2

div     : divVectors     c = b div 3

dot     : dotProduct     c = a dot b

cross     : crossProduct     c = a cross b

Useful features:

clear() - clear all (console and variables)
clear()

color(nameVector, "color") - change color of vector
color(a, "blue")
