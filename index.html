3D Math Calculator

Instruction:

Create vectors:
a = vec3(-100, 10, 100)
b = vec3(200, 200, 12)

Operations:

+     : addVectors     c = a + b

-     : subVectors     c = a - b

*     : mulVector     c = a * 2

/     : divVectors     c = b / 3

⋅     : dotProduct     c = a ⋅ b

x     : crossProduct     c = a x b

Useful features:

clear() - clear all (console and variables)
clear()

color(nameVector, "color") - change color of vector
color(a, "blue")
